# TerraMatrix Home Assistant Add-ons

Home Assistant add-on repository for TerraMatrix.

## Add Repository

Use this repository URL in Home Assistant:

```text
https://github.com/terramatrix-io/terramatrix-ha-addons
```

## Add-ons

- `terramatrix`: TerraMatrix bridge and viewer for Home Assistant.

## Images

The TerraMatrix add-on uses prebuilt GHCR images:

```text
ghcr.io/terramatrix-io/terramatrix-ha-addon-{arch}
```

Supported architectures:

- `amd64`
- `aarch64`
- `armv7`
